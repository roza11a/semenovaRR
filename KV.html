Контрольные вопросы

Что такое «система контроля версий»?

Система контроля версий — это инструмент, который помогает отслеживать изменения в файлах и управлять версиями проекта.

К какому типу VSC относится? В чем главное отличие от других типов?

Git относится к распределенным системам контроля версий. Главное отличие в том, что каждая копия репозитория является полной и независимой.

В чем заключается преимущество использования Git?

Преимущества Git включают возможность работы в оффлайн-режиме, высокая скорость, а также мощные инструменты для слияния и разрешения конфликтов.

Какие команды Git вы знаете?

Некоторые команды Git: git init, git clone, git add, git commit, git push, git pull.

В чем отличие Git и GitHub?

Git — это система контроля версий, а GitHub — это облачный сервис для хостинга репозиториев Git и совместной работы над проектами.
